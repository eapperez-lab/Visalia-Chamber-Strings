# Visalia Chamber Strings Website

React + Vite site for Visalia Chamber Strings.

## Local preview

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel settings

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install

## Ticket link

Replace `LUDUS_TICKET_URL` in `src/main.jsx` with the real Ludus ticket URL.
